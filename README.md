A full-stack web application with secure authentication and role-based access control, built using Spring Boot and React. Users can sign up, log in, and access protected resources based on their role.

Features

User Authentication: Signup and login with JWT (RS256) and BCrypt password hashing

Role-Based Access Control (RBAC): Restrict access to API endpoints based on user roles

Protected APIs: Stateless REST endpoints secured with Spring Security

Frontend: React.js with protected routes and token-based session management

Popup & Navigation Handling: Prevent accidental back navigation and allow logout/save changes

Technologies

Backend: Java, Spring Boot, Spring Security, JWT (RS256), RBAC, REST APIs, Hibernate

Frontend: React.js, React Router, Axios, CSS Modules

Database & Tools: H2 / MySQL, Git, Maven, Postman

Setup

Clone the repository

Run the Spring Boot backend (mvn spring-boot:run)

Run the React frontend (npm install & npm start)

Access the app at http://localhost:3000
